ReadMe-3D Terrain

Download AeonicTextuerPack.zip
		 ExtraTerrainAssets.zip
	Import both the files
Adjust the height of the terrain by clicking on the hill Icon
Use the brush to paint the terrain with textuers
Add Textuer by clicking on the edit texuer and click on Add textuer
Click on the tree icon to add trees 
Click on Edit textuer to add new tree textuer
If you want to add colliders to trees then goto the tree textuer folder drag and drop the tree to the hierarchy
and add collider from the Inspector
